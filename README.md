# Portfolio - Next.js + TypeScript starter

This repository contains a minimal Next.js 14 + TypeScript starter scaffold.

Getting started

1. Install dependencies:

```powershell
npm install
```


2. Run dev server:



```powershell
npm run dev
```

Files created

- `package.json` - project definitions and scripts

- `tsconfig.json` - TypeScript config
- `next.config.js` - Next.js configuration
- `pages/_app.tsx` and `pages/index.tsx` - app entry and homepage
- `styles/globals.css` - global CSS
- `public/` - static assets

Notes

- Type declarations come from `next` and `react` packages. Install dependencies before opening the project in the editor to avoid type errors.
